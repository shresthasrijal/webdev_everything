q1) What Is a String in JavaScript, and What Is String 
Immutability?

- string in JavaScript is simply a piece of text
- a primitive data type:

- can be made with single or double quotes...start and end  
with the same type of quote.

= Immutability means unchangeable after creation.
= in js once u make a string, u cant change its individual  
    characters
    (eg)
    let name = "Jessica";
    name[0] = "Q"; // ❌ does nothing

Instead u must create a new string..
    let name = "Jessica";
    name = "Quincy";

Even operations like replacing or modifying text don’t 
    change the original string—they return a new string:

    let word = "hello";
    let newWord = word.toUpperCase();

    console.log(word);     // "hello" (unchanged)
    console.log(newWord);  // "HELLO"

SO takeaway....
- Strings store text.
- They are primitive and immutable.
- You don’t modify a string—you replace it or create a new  
    one.

q2) What Is String Concatenation, and How Can You 
    Concatenate Strings with Variables?

In JavaScript, string concatenation means joining text 
together.

3 main ways to do it....

i- + operator: Joins strings together.
    "John" + " " + "Doe" → "John Doe"

ii - += operator: Adds text to an existing string.
    let g = "Hello";
    g += ", John!";
    // "Hello, John!"

iii - concat() method: Joins multiple strings using a method.
"Hello".concat(" ", "World") → "Hello World"

Strings in JavaScript are immutable, so every concatenation 
creates a new string, not a modification of the original one.

q3) What Is console.log Used For, and How Does It Work?

console.log() is a simple yet powerful tool used to display 
messages or output information to the browser's console. 
It's mostly used by developers to debug and inspect code 
while working on their programs.

You can use console.log() to log text or variables to the 
console and ensure your code is running correctly.

    console.log("Hello, world!");

    let num = 5;
    console.log(num); // 5

    let name = "Alice";
    console.log("Hello, " + name + "!"); // Hello, Alice!

You can also pass in multiple values to console.log() 
    separated by commas. For example:

    let name = "Alice";
    let age = 25;
    console.log("Name:", name, "Age:", age);
    // Name: Alice Age: 25

The console.log() method helps you monitor your code as it 
runs, making it easier to spot mistakes and understand how 
your program behaves.
