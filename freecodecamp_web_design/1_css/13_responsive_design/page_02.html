Introduction

JavaScript is a lightweight, cross-platform scripting language used to control behavior in web environments.

It includes built-in objects like Array, Date, and Math, along with core features such as operators and control structures.

  • Client-side JavaScript interacts with the browser and DOM, handling user actions like clicks and form input.
  • Server-side JavaScript works with databases, files, and server processes.
What you should already know

This guide assumes basic knowledge of:

  • The Internet and how the web works
  • HTML fundamentals
  • Basic programming concepts
JavaScript and Java

JavaScript and Java share similar syntax but are very different languages.

JavaScript uses dynamic typing and a prototype-based model, unlike Java’s class-based system.

It is flexible and does not require strict declarations for variables or functions.

Hello world

Write your first JavaScript function below:

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");

Run the code to see the result in your browser.

Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($); Also case sensitive!

You can use ISO 8859-1 or Unicode letters such as å and ü in identifiers. You can also use the Unicode escape sequences as characters in identifiers.

Declaring variables

You can declare a variable in three ways:

With the keyword var. For example,

var x = 42.

This syntax can be used to declare both local and global variables.

By simply assigning it a value. For example,

x = 42.

This always declares a global variable. It generates a strict JavaScript warning. You shouldn't use this variant.

With the keyword let. For example,

let y = 13.

This syntax can be used to declare a block scope local variable. See Variable scope below.
