#) WORKING WITH CSS GRIDS

1) What is CSS grid, and how does it differ from flexbox?

    ans:
    CSS Grid is a powerful layout system that allows web 
    developers to create complex and responsive web page layouts 
    with ease.

    Imagine ur arranging furniture in a room, css grid here acts 
    like an invisible grid on ur floor that helps u position 
    everything precisely where u want.

    When we build websites, we often need to arrange different 
    elements on the page. Before css grid, it was tricky with 
    flexbox, but with grid u simplify the process via dividing 
    webpage into rows and columns!

    (eg)
        <div class="container">
            <div class="item">Item 1</div>
            <div class="item">Item 2</div>
            <div class="item">Item 3</div>
            <div class="item">Item 4</div>
            <div class="item">Item 5</div>
            <div class="item">Item 6</div>
        </div>

    With grid box u can arrange it however u want.....

        .container {
            display: grid;
            grid-template-columns: 1fr 1fr 1fr;
            grid-gap: 20px;
        }

        .item {
            background-color: lightgray;
            padding: 20px;
            text-align: center;
            border: 1px solid #ccc;
        }

    This tells browser to create a grid with three equal-width 
    columns, that's what the 1fr 1fr 1fr means, and we're adding 
    a 20-pixel gap between each grid item.

    Why not just flexbox?
    -> Flexbox is one dimensional while grid is 2d... flexbox is 
    great for laying things ina  single row or column, whilst 
    grid excels at creating layouts with both rows and columns.

    Flexbox is content-first, meaning it adjusts the layout 
    based on the content. Grid, on the other hand, is 
    layout-first, allowing you to create the layout and then 
    place items into it. Grid gives you more precise control 
    over placement. You can tell an item exactly which row and 
    column to occupy.
    (eg)
        .container {
            display: flex;
            justify-content: space-between;
        }

    This creates a flex container where the items are spaced 
    evenly along the main axis.

    Both grid and flexbox are useful and best layouts use 
    both.... but usually grid is for overall page layour, and 
    flexbox for aligning items within each grid area...

2) How Can You Create Flexible Grids with the fr Unit?

    ans:
    Let's start with this HTML
        <div class="grid-container">
            <div class="col"></div>
            <div class="col"></div>
            <div class="col"></div>
            <div class="col"></div>
        </div>

    Now CSS:

        html, body {
            width: 90%;
            height: 50%;
        }

        .grid-container {
            display: grid;
            grid-template-columns: 25% 25% 25% 25%;
            gap: 15px;
            background-color: darkgray;
            height: 100%;
        }

        .col {
            background-color: darkslateblue;
        }
    
    here grid-template-columns is used to set the size for each column. Then the 
    gap properaty is used to create space around each column!

    Above we used percentages, but we can also use fr units. 
    Fr unit is a fractional unit which represents a fraction of the space for the 
    grid container.
    (above  css example in fr units)

        .grid-container {
            display: grid;
            grid-template-columns: 1fr 1fr 1fr 1fr;
            gap: 15px;
            background-color: darkgray;
            height: 100%;
        }

    Each column will take up one fraction of the available space. Since there are f
    our columns, each column will have an equal share of the space available in the 
    grid container.

    As you start to build your grid layouts, you will find yourself wanting to use 
    fr units more often because they provide a flexible, proportional way to 
    distribute space, allowing you to create responsive layouts that adapt to 
    varying screen sizes without needing to manually adjust pixel values.

3) How Can You Create Gaps Between Tracks in a Grid?

    ans:
    In css grid, a track is the space between two grid lines. They form the rows 
    and columns of the layout.

    To control spacing between grid items, use:
        - column-gap
        - row-gap
        - gap

    html for examples:
        <div class="grid-container">
            <div></div>
            <div></div>
            <div></div>
            <div></div>
        </div>

    i> Column gap example:
        
        .grid-container {
            display: grid;
            height: 100px;
            grid-template-columns: 1fr 1fr 1fr 1fr;
            column-gap: 10px;
        }

        .grid-container div {
            background-color: darkblue;
        }

    - Accepts values like px, em, %, or normal
    - normal = 0 in grid layouts

    ii> Row Gap Example

        .grid-container {
            display: grid;
            grid-template-columns: 1fr 1fr;
            column-gap: 10px;
            row-gap: 30px;
        }

        .grid-container div {
            height: 100px;
            background-color: darkblue;
        }
    - Creates two rows with spacing between them

    iii> Gap (Shorthand) Example

    - Use gap to define both row and column spacing.
        .grid-container {
            display: grid;
            grid-template-columns: 1fr 1fr;
            gap: 30px 10px;
        }

        .grid-container div {
            height: 100px;
            background-color: darkblue;
        }

    - gap: 30px 10px;
        30px → row gap
        10px → column gap
    - If only one value is given → applies to both
    - Supports px, em, %, calc() (not fr)

    row-gap, column-gap, and gap give you flexible control over spacing in 
    grid layouts, helping you create clean and well-structured designs.

4) How can you repeat track listings in a grid layout?

    ans:
    Previously we used grid-template-columns property and setting the value 
    to a few fractional units.
    This is completely valid, but there is an easier way to repeat a section 
    or all ur track listings!

    The repeat() function is used to repeat a section or all of the tracks 
    for columns or rows. This function takes in a repeat count and the tracks 
    you wish to repeat.

    (eg)
        <link rel="stylesheet" href="styles.css">
        <div class="grid-container">
            <div class="box"></div>
            <div class="box"></div>
            <div class="box"></div>
            <div class="box"></div>
        </div>

        .grid-container {
            display: grid;
            grid-template-columns: repeat(4, 1fr);
            column-gap: 10px;
        }

        .box {
            width: 100px;
            height: 100px;
            background-color: darkblue;
        }

    Same as grid-template-columns: 1fr 1fr 1fr 1fr; 

    The repeat() function will accept any valid pattern that you can use for 
    rows or columns.
    example using the repeat() function to set the first and third columns to 
    20 pixels and the second and fourth columns to one fractional unit.

        <div class="grid-container">
            <div class="box"></div>
            <div class="box"></div>
            <div class="box"></div>
            <div class="box"></div>
        </div>
    
        .grid-container {
            display: grid;
            grid-template-columns: repeat(2, 20px 1fr);
            column-gap: 10px;
        }

        .grid-container div {
            height: 100px;
            background-color: darkblue;
        }

    Sometimes, you might opt to write out each individual value instead of 
    using the repeat() function. But there are times when this function comes 
    in handy, especially when you want to repeat a particular pattern for a 
    track listing.

5) Differece between implicit and explicit grid?

    ans:
    An implicit grid in CSS Grid is created automatically by the 
    browser when there are more items than the explicitly defined rows 
    or columns, or when items are placed outside the defined grid.

    You control these automatically generated rows and columns using 
    grid-auto-rows and grid-auto-columns.

    Whilst:
    an explicit grid is manually defined using grid-template-rows and 
    grid-template-columns.

    Explicit grid = planned structure.
    Implicit grid = automatically expanded structure.


    If you define only 2 columns but add 6 items, the browser creates new rows 
    automatically to fit the extra items—this is the implicit grid in action.

        <div class="grid-container">
            <div class="grid-item">Item 1</div>
            <div class="grid-item">Item 2</div>
            <div class="grid-item">Item 3</div>
            <div class="grid-item">Item 4</div>
            <div class="grid-item">Item 5</div>
            <div class="grid-item">Item 6</div>
        </div>

        .grid-container {
            display: grid;
            /* Only 2 explicit columns */
            grid-template-columns: 100px 100px; 
        }

        .grid-item {
            background-color: burlywood;
            border: 1px solid orangered;
            padding: 0.5rem;
            margin: 0.5rem;
        }

    Differences:
    - Explicit grid:
        fixed, predictable, requires planning.
    - Implicit grid:
        flexible, adapts automatically, useful for dynamic or unknown content.
